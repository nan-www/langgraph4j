# What is a README 

README.md files are for humans: 
* quick starts, 
* project descriptions 
* contribution guidelines.

# Structure

a README.md must be structured in, at least, the following paragraphs

* Getting started
* Introduction
* Core Features & Benefits
* Core Concepts Explained
* Project Structure
* Installation
* A Simple Example
* References
